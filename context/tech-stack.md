# Technology Stack

- **Language**: TypeScript
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Canvas**: HTML5 Canvas for the hex grid and token rendering (not DOM-based — too many elements for smooth drag)
- **State Sync**: BroadcastChannel API for cross-tab communication
- **Persistence**: localStorage
- **Testing**: Vitest
- **No backend** — pure client-side SPA
